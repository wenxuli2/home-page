快捷键

移动行：alt+up/down
格式化代码 shift+alt+f

自定快捷件：【文件】-【首选项】-【键盘快捷键】

小写字母=>大写字母 : shift+alt+x
大写字母=>小写字母 : shift+alt+y
首字母小写=>大写 : shift+alt+u

vscode 插件

px to rem
右击-扩展设置-Px-to-rem: Px-per-rem
Number of pixels per 1rem.
    可以设置当前rem与px的比例值
px to em
右击-扩展设置-Px To Em: Root Font Size
    可以设置当前em与px的比例值
