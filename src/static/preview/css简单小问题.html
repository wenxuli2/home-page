用css改变鼠标箭头

[用css改变鼠标箭头](http://www.zzsky.cn/build/content/23.htm)
一、在<body>与</body>之间加入如下代码（这里选择cursor（文本）作为对象，还可以自己改为其他的，如link等）：
    <span style="cursor:*">文本或其它页面元素</span>
二、把*换成15种效果的一种：
    hand      手形
    crosshair      十字型
    text      文本光标
    wait      等待效果
    default      默认效果
    help      问号
    e-size      向右的箭头
    ne-resize      向右上的箭头
    nw-resize      向左上的箭头
    w-resize      向左的箭头
    sw-resize      左下的箭头
    s-resize      向下的箭头
    se-resize      向右下的箭头
    auto      系统自动给出的效果

伪类:和伪元素::的区别

伪类: 
就是这个元素的一个状态 比如 :active :checked    :empty, 都是当这个元素的一个状态.
伪元素:
你需要创建一个新的元素才能完成的样式,也就是说,如果你不用伪元素,你需要信创建一个元素才能完成这样的效果;
比如::after 和::before作用是在元素前后添加内容和样式, 对于伪元素 css提供了一个content属性 可以在css中对伪元素添加内容
