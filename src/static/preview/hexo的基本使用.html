常见问题

1.hexo使用theme出现“ {% extends ‘_layout.swig‘ %} {% import ‘_macro/post.swig‘ as post_template %}“问题
    原因是hexo在5.0之后把swig给删除了需要自己手动安装
    npm i hexo-renderer-swig
2.const { Component } = require('inferno'); const classname = require
    主题插件冲突，建议新的项目使用
